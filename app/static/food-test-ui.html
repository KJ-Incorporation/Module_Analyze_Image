Weighty Food Test UI

Tester l'analyse food directement depuis FastAPI.

Cette page envoie des images de repas en multipart/form-data vers un endpoint relatif du backend. Elle est faite pour tester rapidement le JSON retourne par POST /analyze-food.

Endpoint cote front

/analyze-food

Le chemin est defini dans le JavaScript et reste facile a modifier si besoin.

Formulaire

Les champs vides ne sont pas envoyes. Les images sont ajoutees avec FormData.
